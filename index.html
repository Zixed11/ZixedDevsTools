About Us

Zixed Tools is a Unity Asset Package that gives you scripts for your Gorilla Tag fangame, and some other useful assets! 100% free to use.

Guide

To get started with Zixed Tools, you want to download the ZixedTools.unitypackage file from this link github.com/Zixed11/ZixedToolsDownload. You then want to import it into your unity project, and at the top you should notice something different, it will say "ZixedDev's Tools" and then you should be all set up
